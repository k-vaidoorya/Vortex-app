## Installation

1. **Clone the repository**
    ```bash
    git clone https://github.com/k-vaidoorya/Vortex-app.git
    cd Vortex-app
    ```

2. **Install dependencies**
    ```bash
    # If using npm
    npm install

    # Or if using yarn
    yarn install
    ```

3. **Run the application**
    ```bash
    # For development
    npm start

    # For production build
    npm run build
    ```
